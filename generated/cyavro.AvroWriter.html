cyavro.AvroWriter¶

class cyavro.AvroWriter(str filename, str codec, str schema, size_t block_size=0)¶

Class use for writing pandas dataframes to an avro file.

Parameters:
filename : str

Output filename to be written.

codec : str

Codec to use fot the file. Valid arguments are {‘null’, ‘deflate’, ‘snappy’}. This corresponds to which compression library the encoder will use.

schema : str

Avro Schema as a string.

block_size : size_t

Some of the compression libraries allow specifying what the block size to use is. This specifies the block size in bytes.

Examples

>>> writer = AvroWriter('out.avro', 'null', avroschema)
>>> writer.write(df)
>>> writer.close()

Methods

close(self) Closes the writer object and flushes the file.
write(self, df) Writes a dataframe to the file created.
